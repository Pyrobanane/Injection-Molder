# Injection-Molder

Hey fellow Makers!

This is my open source DIY Injection Molding Machine (made out of Old Ender 3's)

## Bill of Materials (Incomplete)

### Pneumatics
- 2x 100mm Ø32mm Pneumatic Cylinder
- 1x 5/2 Way Pneumatic Valve
- Pneumatic Tubing
- 7x Quick Connectors

### Power Electronics
- 24V PSU
- 5V PSU (or Buck Converter)
- SSR Relay
- 50W AC Ceramic Band Heater
- WAGO Spring Clamps
- Flexible Wire

### Low Voltage ELectronics
- ESP 32 C3 Supermini
- Active Buzzer
- Rotary Encoder
- 3x Push Buttons
- JST Connectors
- NTC 3950 100K
- 100k Resistor
- 2x TM1637 Display
- Wire
- Perf Board PCB
- Pin Headers

### Hardware Components
- M4 T-Nuts
- M4 Screws
- M2 Screws
- M2 Heat Set Inserts
- M5 Grub Screws
- 3* 2040 Aluminium Extrusion (Ender 3 Gantry)
- 2* 4040 Aluminium Extrusion (Ender 3 Base)
- 1* 2020 Aluminium Extrusion (Ender 3 Top Bar)
- some kind of round metal bar (I used brass)
- Ender 3 Nozzle
- Aluminium Flat Stock


### Tools
- Lathe for the Injection Chamber (i think you might be fine with a Pipe of some kind)
- Desktop CNC (i think you might be fine with a Drill Press)
- 3D Printer
- Crimping Tool
- Screwdrivers
- Soldering Iron
- Pliers

I used **PLA** Filament but i would reccomend you use something better, but it works **:)**
